## ğŸš€ Getting Started

### Prerequisites
- Node.js (v16+)
- Python (v3.8+)
- MongoDB running locally or a connection string

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Heavenly-Demon-0835/LinkFluence.git
   cd LinkFluence
   ```

2. **Backend Setup**
   ```bash
   cd backend
   python -m venv venv
   # Windows
   venv\Scripts\activate
   # Mac/Linux
   # source venv/bin/activate
   pip install -r requirements.txt
   python app.py
   ```

3. **Frontend Setup**
   ```bash
   cd frontend
   npm install
   npm run dev
   ```
